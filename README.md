# pager2

update to [pager](https://github.com/dannylindquist/pager)

### development

create a `.env` file in root.

```
//.env
COOKIE_SECRET=MY_COOKIE_SECRET
```

may require node > 14  
testing done on node@16

```javascript
npm install
npm run dev
```

### prod

`npm run build`
`npm run start`

### deployment

hosted on [fly](https://fly.io)

[pager.fly.dev](https://pager.fly.dev)
